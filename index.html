AimBot

Description:

Aim the closest player about you

Commands:

Mouse3 -- Activate / Desactivate

.team -- Toggle the team's members's aim

.autoshot -- Toggle the automatic shot when a player is detected

»AimBot

AimAssist

Description:

Aim the closest player about your mouse

Commands:

Mouse3 -- Activate / Desactivate

.r [NUMBER] -- Set the detection radius

.team -- Toggle the team's members's aim

.autoshot -- Toggle the automatic shot when a player is detected

Download:

»AimAssist
